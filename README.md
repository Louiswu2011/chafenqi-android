# 中二节奏/舞萌DX查分器
**QQ交流群号：348605187**

用Jetpack Compse实现的轻量级查分器，支持Android 9及以上。

[iOS端](https://github.com/Louiswu2011/chafenqi)

## 主要功能
- [x] 基本信息查询
- [x] Rating/B30/R10计算
- [x] 搜索曲目
- [x] App内查看谱面（仅Expert及以上）
- [x] 个人最佳成绩查询
- [x] 分数上传
## 使用方法
### QQ群（推荐）
请到Q群内获取最新版本apk！
### Github Release（更新较慢，不推荐）
1. 下载Release中的apk文件
2. 安装
3. 开始查分！
## 注意事项
- 偶发闪退
- 暂不支持中二WE谱面
## 编译项目
1. Clone到本地
2. 用Android Studio打开编译
